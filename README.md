## 功能概览

| 功能模块 | 支持操作 |
|---------|---------|
| **场景管理** | 场景列表、切换、新建、删除、重命名；双击查看场景内各源项；单击预览该场景截图 |
| **音频混音台** | 自动列出所有音频源；音量滑块（0–100%）、dB 实时显示、静音切换、VU 电平表 |
| **录制控制** | 开始/停止/暂停/恢复录制；录制目录配置；录制时码实时显示 |
| **推流控制** | 一键推流/停止；实时推流时码；字幕发送（CEA-608） |
| **虚拟摄像头** | 一键开启/关闭虚拟摄像头 |
| **转场控制** | 转场效果下拉选择；时长微调（0–10000ms）；Studio 模式开关；T-Bar 手动拖拽 |
| **滤镜管理** | 选择任意输入源；查看/启用/禁用/删除滤镜；一键应用内置滤镜预设（美颜、黑白、HDR 等） |
| **回放缓冲** | 启动/停止缓冲；一键保存最新 N 秒片段 |
| **截图** | 对任意输入源实时截图；缩略图预览；一键保存到本地 |
| **实时统计** | FPS、CPU、内存、丢帧率大数字卡片展示；OBS 版本信息；状态快照保存/恢复 |
| **实时预览** | PROGRAM（节目）和 PREVIEW（预览）双画面；CUT / FADE / 🌑 黑场 按钮 |
| **操作日志** | 左下角实时日志窗口，带时间戳、颜色分级（INFO/SUCCESS/WARNING/ERROR） |
| **状态栏** | 连接地址、当前场景、录制时码、推流时码、FPS、CPU 一览 |

## 系统要求

| 组件 | 最低版本 | 说明 |
|------|---------|------|
| **OBS Studio** | 28.0.0 | 内置 obs-websocket 5.0 |
| **Python** | 3.9 | 推荐 3.11+ |
| **obsws-python** | 最新版 | OBS WebSocket 通信库 |
| **ttkbootstrap** | 1.10+ | GUI 框架（Dark 主题） |
| **Pillow** | 9.0+ | 图片处理和预览显示 |

> **Windows / macOS / Linux** 均支持，预览截图功能需要 OBS 28+。

## 安装

### 1. 安装 Python 依赖

```bash
pip install obsws-python ttkbootstrap pillow
```

### 2. 启用 OBS WebSocket 服务

1. 打开 **OBS Studio**
2. 菜单栏 → **工具** → **obs-websocket 设置**
3. 勾选 **启用 WebSocket 服务器**
4. 记录生成的**密码**（建议开启密码保护）
5. 确认端口（默认 `4455`）

### 3. 克隆 / 下载项目

```bash
git clone <本仓库地址>
cd obs-console
```

## 界面说明

### 连接工具栏

位于窗口顶部，填写连接信息后点击 **连接**：

| 字段 | 默认值 | 说明 |
|------|-------|------|
| Host | `localhost` | OBS 主机地址（远程控制填对方 IP） |
| Port | `4455` | obs-websocket 监听端口 |
| Password | 空 | 与 OBS WebSocket 设置中的密码一致 |

连接成功后底部状态栏变绿，所有标签页自动刷新数据。

---

### 左侧预览区

左侧竖向分为三个区域：

#### 双画面预览

- **PROGRAM**（上）：当前正在播出的节目画面，约 500ms 刷新一次
- **PREVIEW**（下）：Studio 模式下的预览画面；单击场景列表时也会在此显示该场景截图

#### 快速操作按钮

| 按钮 | 功能 |
|------|------|
| **CUT** | 硬切：立即将预览场景切换为节目场景（无转场效果） |
| **FADE** | 淡切：使用当前转场效果将预览场景切换为节目场景 |
| **🌑 黑场** | Toggle：淡出到纯黑场景 / 从黑场淡入恢复。用于节目暂停或广告插播场景 |

#### 操作日志

左下角日志窗口实时显示所有操作记录，带时间戳和颜色：

| 颜色 | 等级 | 典型内容 |
|------|------|---------|
| 灰色 | INFO | 连接成功、场景切换 |
| 绿色 | SUCCESS | 录制开始、推流启动 |
| 黄色 | WARNING | 转场未找到、连接中断 |
| 红色 | ERROR | 操作失败、崩溃详情 |

### 🎬 场景标签页

左右分栏布局：

**左侧 — 场景列表**

- 显示 OBS 中所有场景，高亮当前节目场景
- **单击**：在左侧 PREVIEW 预览区显示该场景截图（不切换节目）
- **双击**：切换到该场景（切换节目），同时在右侧加载该场景的源项列表
- **右键菜单**：切换 / 重命名 / 删除
- 按钮：**切换 / 新建 / 删除 / 重命名 / 🔄 刷新**

**右侧 — 场景项（源列表）**

双击场景后显示该场景内的所有源：

| 列 | 说明 |
|----|------|
| 名称 | 源的名称 |
| 类型 | 源类型（monitor_capture、image_source 等） |
| 启用 | ✓ 已启用 / ✗ 已禁用 |

操作按钮：**启用/禁用**（Toggle）、**删除项**

### 🎚 音频标签页

自动检测 OBS 中支持音频的输入源（跳过纯视频源），每路音频显示一行：

| 控件 | 功能 |
|------|------|
| 名称标签 | 输入源名称（最多 22 字符） |
| 音量滑块 | 0–100%，拖动实时同步到 OBS |
| dB 标签 | 对应 dB 值实时显示 |
| 🔊 / 🔇 | 切换静音状态 |
| VU 电平表 | 绿/黄/红分段显示当前电平 |

点击 **🔄 刷新** 重新扫描音频源列表。

### ✨ 滤镜标签页

1. 顶部下拉框**选择输入源**，点击 🔄 加载该源的滤镜列表
2. 列表显示该源上所有滤镜（名称、类型、启用状态）
3. 操作按钮：**启用/禁用**、**删除**、**刷新**
4. **预设区**：选择内置预设后点击**应用**，一键添加并启用该滤镜

内置预设包括：美颜-轻度、美颜-中度、美颜-强度、HDR感、电影感、夜间模式、复古胶片、高对比黑白。

## 底层 Python SDK

本项目同时包含 **`obs_controller.py`**，是对 obsws-python 的完整封装 SDK，可单独引入到其他 Python 项目中使用：

```python
from obs_controller import OBSController

with OBSController(host="localhost", port=4455, password="your_password") as obs:
    print(obs.get_current_scene())     # 获取当前场景
    obs.set_current_scene("主场景")   # 切换场景
    obs.set_input_volume("麦克风", volume_db=-10.0)  # 设置音量
    obs.fade_to_black()                # 淡出到黑场
```

SDK 支持 30+ 功能模块，详见 [docs/obs_controller.md](README_obs_controller.md)（原 SDK 说明文档）。

## 常见问题

**Q：点击连接后报"连接超时"？**

- 确认 OBS 已打开，且 obs-websocket 服务已启用
- 确认 Host / Port 和 OBS 中的设置一致
- 如果是远程控制，确认对方防火墙已放行 4455 端口

**Q：音频混音台显示为空？**

- 确认 OBS 中有添加了音频输入源（如麦克风、桌面音频）
- 点击 🔄 刷新重新扫描

**Q：预览画面一直显示"连接中"或黑屏？**

- 预览依赖 `GetSourceScreenshot` 接口，需要 OBS 28+
- 检查 OBS 是否正在运行并有活跃场景

**Q：黑场按钮没有淡入淡出效果？**

- 需要 OBS 中存在"Fade"（英文版）或"淡入淡出"（中文版）转场
- 在"🎞 转场"标签页检查转场列表

**Q：推流字幕不显示？**

- 字幕（CEA-608）需要推流平台支持，部分平台不显示嵌入字幕
- 确认推流已启动

## 注意事项

1. **OBS 版本**：必须使用 OBS Studio 28.0+，旧版本的 obs-websocket 4.x 协议不兼容
2. **密码安全**：生产/直播环境建议开启 obs-websocket 密码，避免局域网内被他人控制
3. **预览性能**：双画面预览每 500ms 截图一次，OBS 负载轻微增加；低配置机器可适当降低刷新频率
4. **线程安全**：控制台内部已实现 ReqClient 线程安全锁，多个操作可并发执行而不会导致响应错乱
5. **事件自动刷新**：场景的新建/删除/重命名会通过 OBS 事件自动同步到场景列表，无需手动刷新
6. **音频源过滤**：音频混音台只显示支持音量控制的音频源，纯视频源（文本、图片、显示器捕获等）自动排除
